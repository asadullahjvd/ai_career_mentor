# ai_career_mentor
🤖 AI-powered career guidance system that analyzes your resume and recommends career paths, identifies skill gaps, generates a learning roadmap, and finds live job opportunities using LLaMA3, FAISS, and JSearch API.

Note:
## Dataset
This project uses the [Job Descriptions Dataset](https://www.kaggle.com/datasets/bcsf24m006asadullah/job-desciptions) hosted on Kaggle.
